# OneCard System

Smart Student Card & School Management System for Jinja Senior Secondary School.

## Overview

OneCard replaces monthly paper card printing with permanent PVC cards featuring QR codes. Staff scan cards to instantly check fees, mark attendance, track movement, and control meal access.

## Tech Stack

Python 3 + Django 5.0 | MySQL 8.0 | HTML5, CSS3, JavaScript | Chart.js

## Features

- Student Import from existing database
- QR Cards with version tracking
- Card Design with color-coded templates
- Day/Hostel identification
- Scanner (Attendance, Balance, Pass-Out, Meals)
- Auto-Attendance on every scan
- Late Tracking with admin cutoff time
- Fee Management (Day/Hostel)
- Meal Access Control with balance thresholds
- Movement Tracking (pass-out/return)
- Reports with Excel/PDF export
- 5 User Roles with permissions
- 31 Mobile Templates
- Rate Limiting & Security
