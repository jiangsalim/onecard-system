## 1. WHAT IS ONECARD?

OneCard replaces monthly paper student cards with permanent PVC cards containing QR codes. When scanned, the system instantly shows the student's photo, name, class, and fee balance. It automatically records attendance, controls meal access, and tracks student movement.

**One card. Entire school life. No monthly reprinting.**

---

## 2. KEY FEATURES

| Feature | What It Does |
|---------|-------------|
| **Permanent Student Cards** | One PVC card with QR code — lasts until graduation |
| **Instant Fee Verification** | Scan card → See balance in 2 seconds |
| **Automatic Attendance** | Recorded every time a student is scanned anywhere |
| **Meal Access Control** | Day scholars get lunch only; Hostel gets all meals; Prevents double-serving |
| **Movement Tracking** | Records when students leave and return; Shows who is outside |
| **Gender Analytics** | Attendance reports filtered by Male/Female per class, stream, and school |
| **Teacher Dashboards** | Weekly attendance charts, gender distribution, absent student alerts |
| **Staff Messaging** | In-app chat between teachers, bursar, and administration |
| **Card Design & Printing** | Color-coded cards by class level; Print with photo, QR code, and barcode |
| **Fee Reports** | Filter by Cleared/Not Cleared; Export to Excel or PDF |
| **Mobile Friendly** | Works on phones, tablets, and computers |
| **Role-Based Access** | Super Admin, Admin, Bursar, Gate Staff, Class Teacher — each with specific permissions |

## 3. HOW STUDENT DATA IS MANAGED

We offer TWO options for managing your student and payment data. Please choose one:

---

### OPTION A: STANDALONE SYSTEM (Recommended)

**OneCard becomes your complete school management system.**

| Aspect | Detail |
|--------|--------|
| **How students get in** | You provide an Excel file with all students (names, classes, streams, categories). We import them once into OneCard. |
| **How payments get in** | Every week, the bursar uploads payment reports from MTN Mobile Money or Airtel Money as an Excel file. |
| **Where data lives** | Hosted on a secure cloud server (Render). Accessible from anywhere. |
| **Internet needed?** | Only when uploading files. Scanning and day-to-day use works without internet. |
| **IT support needed** | Minimal — bursar handles file uploads. |
| **Setup time** | 1 week from receiving your student data. |
| **Monthly cost** | ~$14 (approximately 50,000 UGX) for cloud hosting. |

**Best for schools that:** Want a fresh start with one simple system and minimal IT dependency.

---

### OPTION B: CONNECTED SYSTEM

**OneCard connects to your existing school database.**

| Aspect | Detail |
|--------|--------|
| **How students get in** | OneCard reads live from your current student database. |
| **How payments get in** | OneCard reads live from your payment records. |
| **Where data lives** | Your existing database stays on your server or is moved to the cloud. |
| **Internet needed?** | Yes — your database must be accessible online 24/7. |
| **IT support needed** | Moderate — requires someone to maintain the database connection and server. |
| **Setup time** | 2-3 weeks (includes database connection setup and testing). |
| **Monthly cost** | ~$14 for cloud hosting + initial IT setup fee. |

**Best for schools that:** Want to keep their current student management system and have reliable IT support.
